## 特性

- **浅色 & 深色模式** — 包含两套独立主题文件
- **中文优化** — Inter + JetBrains Mono + 中文字体回退

## 安装

1. [**下载**](https://github.com/qishaoyumu/typora-blue-topaz-theme/archive/refs/heads/master.zip) zip 包并解压，或克隆本仓库
2. 打开 Typora，进入 `偏好设置 > 外观 > 打开主题文件夹`
3. 将以下文件复制到主题文件夹中：
   - `blue-topaz.css`
   - `blue-topaz-dark.css`
   - `blue-topaz/` 文件夹（字体）
4. 重启 Typora，在主题菜单中选择 **Blue Topaz** 或 **Blue Topaz Dark**

## 推荐字体

主题已内置 **Inter**（正文）和 **JetBrains Mono**（代码）。为获得最佳中文显示效果，建议安装：

- [霞鹜文楷 GB / LXGW WenKai](https://github.com/lxgw/LxgwWenKai)
  - macOS: `brew install font-lxgw-wenkai`
  - Windows: 从 [GitHub Releases](https://github.com/lxgw/LxgwWenKai/releases) 下载
  - Linux: `sudo apt install fonts-lxgw-wenkai` 或从 Releases 下载

未安装时，主题会自动回退到系统字体（苹方 / 微软雅黑 / Noto Sans CJK）。
